Plastic Pollution

The plastic pollution data set contains information from the organization, Break Free From Plastic, and their audits on plastic pollution and the efforts to clean it up within different countries. It contains data from audits in 2019 and 2020, in addition to the parent company from where the pollution originated from. The plastic is specifically classified by recycling codes.

Looking at the cleaning that has been done to the data so far, it is clear that from the cleaning script, the columns have been set to have values of either values of characters or doubles. This applies for both 2019 and 2020. Furthermore, for 2019, there has been an str_replace operation conducted to better set the names from the file which it was collected from. Additionally, there has been a mutate operation conducted to create new variables of pp and ps, with if else statements to handle NA values within the variable pp_2.

Initial Research Questions

  1. Which companies had the highest grand total number of plastics thrown out in 2019 and 2020?

  2. Which countries had clean up efforts that found the largest grand total of plastics in 2019 and 2020?

  3. Does the amount of plastic waste within countries that are part of the Paris Climate Agreement significantly differ from countries who are not a part of the agreement?

  4. Do countries with a higher GDP tend to contribute to rising levels of plastic waste?

plastic_per_event <- function(input_country) {
  plastics |>
    filter(country == input_country) |>
    distinct(year, grand_total, num_events) |>
    summarise(
      total_plastic = sum(grand_total, na.rm = TRUE),
      total_events = sum(num_events, na.rm = TRUE)
    ) |>
    mutate(plastic_per_event = total_plastic / total_events) |>
    pull(plastic_per_event)
}

Create a function to pull data from the API

get_gdp <- function(country_code, key = my_key) {
  url <- glue("https://api.api-ninjas.com/v1/gdp?country={country_code};X-Api-Key={key}")
  
  my_data <- safely(fromJSON)(url)
  
  if (is.null(my_data$result)) {
    return(NA)
  } else {
    return(my_data$result |> filter(year %in% c(2019, 2020)))
  }
}

New Data Descriptions

GDP Data Description

The GDP data set has been extracted from the API Ninjas “GDP” API. In this case, the data has been programatically obtained in R by making an API call for each country by its ISO3 country code. Annual Gross Domestic Product (GDP) in U.S. Dollars has been recorded in this data set.

However, for this assignment, the years included have been limited to 2019 and 2020. Therefore, each country will appear twice in this data set corresponding to the two years. The data is in the form of a Tidy Data Frame, and it would later be merged with the plastic data set.

Key variables include:

  • country: ISO3 country code

  • year: year of observation (2019 or 2020)

  • value: GDP (nominal) in U.S. dollars

Renewable Energy Data Description

The renewable energy dataset was sourced from an open-source data set provided on GitHub and is based on World Development Indicators. The data set consists of country-specific information about renewable energy consumption over time.

In this case, the renewable energy data set has been filtered to include only the years 2019 and 2020 as well as only those countries that appear in the plastic data set. This filtering was done using ISO3 codes.

Key variables include:

  • Country Code : ISO3 country code

  • Year: year of observation

  • renewable energy consumption, measured as the percentage of total final energy consumption from renewable sources

This dataset was merged with the GDP and plastics datasets to support comparative analysis across countries.
